What is a Science Gateway?

Science gateway is a collection of tools, applications and data that allows scientist to run their applications having minimum concern of where the actual processing takes place or the underlying complexity of computational resources. Science gateways open up opportunities to access all services in one particular place without worrying where the all services located.And scientists can reuse what other scientists have done.

Science gateways are mainly involved with communities who are interested in specific areas such as bioinformatics, astrophysics, oceanography and geoscience.But such a development costs significant amount of money and time. Thus they should focus on funding, project goals, tools, community engagement, rewards and recognition as described by[1].

Scientist should be able to reuse what other scientists have implemented in the same domain which has become major motivations for building science gateways. There are several places where scientists can consider for access high performance computing. It can be regional grids such as NWICG , Open Science Grid, TeraGrid(Now the project has been extended as XSEDE) and DOE. Most of the science gateways have 3 layers; application layer, middleware layer (SAGA) and resource layer (Teragrid).

Actually Teragrid provides services to several science gateways.

  • Computational Chemistry Grid (GridChem)
  • Linked Environments for Atmospheric Discovery (LEAD)
  • nanoHUB.org
  • Cancer Biomedical Informatics Grid (caBIG)

What they provide?

Science gateways typically provides data sources, set of tools to handle them, computational resources and resources for novel scientists to learn about technologies and other domain related matters. Mostly scientist access sciennce gateways via an increasingly capable web interfaces which provides those set of tools, data sources and learning material. Among the set of tools provided WfMS plays a major role. Scientists can download that and create workflows consuming the high performance computational resources. Scientists can create an account in the portal and start using its resources right away.

What are the geoscience gateways?

Actually GEON provides set of sciennce gateways inclusing the famous Open Topography. Instead of the above mentioned resources thesen provides ability to practice WPS(Web Processing Service) via the portal. And there are attempts to amalgamate sciennce gateways to get the best parts of two science gateways together.

Plus another aspect of science gateways are VO (Virtual Organization)[2]. VO’s are formed to solve a particular problem. Basically involved with solving a small problem by group of people. They need to share information and data within that problem domain with relevant tools associated.
